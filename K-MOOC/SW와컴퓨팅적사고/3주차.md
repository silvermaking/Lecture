# SW와 컴퓨팅적 사고

# 3주차. 인간의 사고체계

## 창의성

- 독창성: 기존의 방식과 다른 새로운 것
- 실현성: 이상적이지 않은 현실적인 것
- 가치: 인류에게 가치를 줄 수 있는 것



### 역량

- 연상력: 사물을 다르게 생각
- 연관력: 개념과 개념을 연관
- 공감력: 이해의 차원을 넘어 고객과 같은 마음을 느끼는 능력

# 2-2. 데이터 나타내기

## 왜 2진법?

- 최초의 컴퓨터 애니악
  - 처음에는 10진수 사용
  - 10자리의 링 카운터를 사용하여 숫자 저장
  - 36개의 진공관이 필요한 복잡한 구조
  - 전압이 불안정한 문제 해결을 위해 2진수를 사용한 회로를 사용

- 간단한 논리회로 설계로 더 신속하고 신뢰성을 유지



## 비트(bit)

- Binary Digit
- 2진수에서 '하나의 자릿수'
- 컴퓨터에서 정보를 저장하는 가장 기본적인 단위



- 바이트(byte): 8개의 비트가 모인 것
